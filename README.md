# OIBSIP Data Science Internship Projects

This repository contains the projects completed as part of the **Oasis Infobyte Data Science Internship (OIBSIP)**.

## 📊 Projects Completed

### 1️⃣ Iris Flower Classification
- Built a machine learning model to classify iris flowers into three species:  
  **Setosa, Versicolor, and Virginica**.
- Used algorithms like Logistic Regression, KNN, Decision Tree, and SVM.
- Evaluated the models using accuracy score and selected the best performing model.

### 2️⃣ Unemployment Analysis
- Analyzed unemployment data using Python.
- Performed data cleaning and exploratory data analysis (EDA).
- Created visualizations to understand unemployment trends and patterns.

### 3️⃣ Car Price Prediction
- Developed a machine learning model to predict car selling prices.
- Performed data preprocessing and feature engineering.
- Trained multiple regression models such as Linear Regression, Decision Tree, and Random Forest.
- Evaluated model performance using R² score.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
